1. Build Your Frontend

Use the Frontend tab to drag components onto your canvas. Select any element to style it in the properties panel on the right.

  • Drag components from the left panel
  • Click to select and style in properties
  • Use breakpoints to design responsive layouts

2. Build Your Backend

Switch to the Backend tab. Drag nodes from the library to create API routes. Connect nodes with wires to define your data flow.

  • Drag-and-drop nodes from the palette
  • Connect ports to wire logic together
  • Every node generates real Express.js code

3. Connect Front to Back

Select a button or form in the Frontend tab, go to the Bind properties tab, and connect it to your backend route.

  • Select a frontend element
  • Open the Bind tab in Properties
  • Choose the backend route to connect

4. Deploy

Click Deploy in the top-right. StateForward builds your HTML/CSS/JS frontend and Node.js/Express backend into a deployable package.

  • Standard HTML + Node.js output
  • Remove StateForward — code stays intact
  • SQLite included, ready to swap for Postgres
